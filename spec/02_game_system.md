# ゲームシステム仕様

## 1. フィールド探索・基本仕様

### 1.1 フィールドの構造
- **視点**: 見下ろし視点（トップビュー）の2Dマップ。
- **画面表示サイズ**: 960 × 544 ピクセル。
- **フィールド全体の大きさ**: 画面表示サイズ（960 × 544）の**面積100倍**（幅10倍 × 高さ10倍：**9,600 × 5,440 ピクセル**、画面100枚分の広さ）。
- **環境・演出**: 夜中の山の中を想定した暗い画面表示。
  - プレイヤーの持つ「懐中電灯」の光が届く範囲（プレイヤーの周囲）だけがかろうじて見えているような視界制限演出を行う。

### 1.2 操作方法
PCでのプレイを想定したキーボードおよびマウス操作。

| 入力デバイス | 操作内容 | 機能 |
| :--- | :--- | :--- |
| キーボード | `W` `A` `S` `D` キー | プレイヤーの移動（上下左右） |
| キーボード | `R` キー | 地図の表示（地図所持時のみ有効） |
| キーボード | `F` キー | 鈴の使用 |
| マウス | 右クリック | アイテムの取得、NPCとの会話 |

## 2. アイテムシステム
フィールドには以下のアイテムがランダムに配置される。いずれも初期配置数は限定されている。

### 2.1 地図 (出現数: 1個)
- **効果**: 
  - 獲得すると、フィールドのマップ全体を表示できるようになり、山の出口（脱出口）の位置がわかるようになる。
  - 獲得次第、画面上に「地図所持ボタン」が表示される。
  - キーボードの `R` キーを押すことで、地図画面を表示することができる。
- **目的**: ゲームクリアのために必須のアイテム。

### 2.2 鈴 (出現数: 3個)
- **効果**: 
  - 使用すると、フィールド上で熊を寄せ付けない効果を発揮する。
  - 獲得するたびに所持数が 1 ずつ増加する。
  - 獲得次第、探索画面（UI）に「鈴の使用ボタン」が表示される。
  - キーボードの `F` キー、または画面上のボタンクリックで使用可能。
- **耐久力（制限・管理仕様）**: 
  - すぐに錆びてしまうため、1つの鈴につき2回使用すると壊れてしまう。
  - **管理仕様（B案）**: 鈴を1個獲得するごとに、全体の「残り使用可能回数」が **+2回** され、共通のカウンターとして加算・管理される。使用するごとにカウンターが 1 減少し、0 になると鈴の効果は使えなくなる。

### 2.3 熊撃退スプレー (出現数: 3個)
- **効果**: 
  - クマとの戦闘画面でのみ使用可能な防衛アイテム。
  - 使用すると、クマを一度だけ噴射して遠ざけ、戦闘を離脱して探索画面に戻ることができる。
- **消費**: 
  - 使用するたびに所持数を1減らす。

## 3. キャラクターとNPC

### 3.1 プレイヤー（主人公）
- 山に迷い込み、懐中電灯を頼りに脱出を目指す。

### 3.2 NPC（遭難者・通行人など）
- **配置数**: フィールド上のランダムな位置に、合計5人ほど配置される。
- **移動**: その場から動かず、直立（固定位置）している。
- **対話**: マウス右クリックで話しかけることができる。
- **会話内容**: プレイヤーの感情を高めるセリフや、アイテムに関するアドバイス、世界観の説明など。
- **必須会話内容**:
  1. 「遠くにいる時は静かにその場を去る方がいい」
  2. 「近くにいる時は熊を見ながらゆっくり後退して！興奮させないのが一番だよ」
  3. 「鈴はクマが寄ってこなくなるがすぐに錆びてしまうため、長くは使えない。」
