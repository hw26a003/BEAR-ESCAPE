# ゲームシステム仕様

## 1. フィールド探索・基本仕様

### 1.1 フィールドの構造
- **視点**: 見下ろし視点（トップビュー）の2Dマップ。
- **画面表示サイズ**: 960 × 544 ピクセル。
- **フィールド全体の大きさ**: 画面表示サイズ（960 × 544）の**面積100倍**（幅10倍 × 高さ10倍：**9,600 × 5,440 ピクセル**、画面100枚分の広さ）。
- **環境・演出**: 夜中の山の中を想定した暗い画面表示。
  - プレイヤーの持つ「懐中電灯」の光が届く範囲（プレイヤーの周囲）だけがかろうじて見えているような視界制限演出を行う。
- **大ブロックの配置方法（自動生成）**: 256×256ピクセルまたは512×512ピクセルの大ブロック（背景パーツ）を、プログラムによってランダムにパズルのように組み合わせてフィールドマップを自動生成する。
- **通行不可領域（衝突判定）**: 木や岩などの障害物がある場所は「通れない場所」として判定する。プレイヤーやクマがその木や障害物に「直接当たった時」に衝突判定（コリジョン）が発生し、それ以上進めないようにする（クマが障害物に当たった場合も同様にその場で進行を阻まれる）。

### 1.2 操作方法
PCでのプレイを想定したキーボードおよびマウス操作。

| 入力デバイス | 操作内容 | 機能 |
| :--- | :--- | :--- |
| キーボード | `W` `A` `S` `D` キー | プレイヤーの移動（上下左右） |
| キーボード | `Shift` キー（長押し） | 走る（ダッシュ） ※スタミナ消費 |
| キーボード | `R` キー | 地図の表示（地図所持時のみ有効） |
| キーボード | `Q` キー | 鈴の使用（簡易マップ同様に探索中に使用可能） |
| マウス | 右クリック | アイテムの取得、生存者（NPC）の救出 |

### 1.3 ダッシュとスタミナシステム
- **ダッシュ動作**:
  - `Shift` キーを長押ししながら `W` `A` `S` `D` キーで移動することで、通常移動（歩き）よりも高速に移動（ダッシュ）できる。
  - 通常移動速度を 100% とした場合、ダッシュ時は **160%** の速度となる。
- **スタミナゲージ**:
  - 画面上（例：プレイヤーの頭上、またはUIの端）に「スタミナゲージ」を表示する。
  - ダッシュ中は、スタミナゲージが徐々に減少する（最大状態から **5秒間** の連続ダッシュでゲージが空になる）。
  - ダッシュを止め（通常移動または静止状態）、キーを離すと、スタミナゲージが徐々に自動回復する（空の状態から全回復まで約 **3秒**）。
- **スタミナ切れペナルティ**:
  - スタミナゲージが 0 （空）になると、スタミナ切れ状態となり、ダッシュができなくなる。
  - スタミナゲージが一定割合（例：30%以上）回復するまで、ダッシュの再実行は不可能となる。

### 1.4 懐中電灯のバッテリー制限
- **制限時間**: 懐中電灯のバッテリー寿命は **10分（600秒）** とする。
- **光の減衰演出**:
  - ゲーム開始時（10分残り）が最も明るく、光の照射範囲（円の半径）が広い。
  - 時間経過に比例して、プレイヤーの周囲を照らす円形の可視範囲（光の半径）が徐々に縮小（減衰）していく。
  - 残り時間 2分（120秒）以下になると、懐中電灯がパチパチと明滅（ノイズ）し始め、極めて狭い範囲しか見えなくなる。
  - 10分が経過するとバッテリーが完全に切れ（完全な暗闇）、プレイヤーの周囲ごくわずか（キャラクターと密着する極小範囲）の視界しか残らなくなる。

### 1.5 キャラクター・アイテムの配置制限
- **出現エリアのルール**: アイテムや生存者、脱出口が「障害物（木など）の内部」に重なって出現しないようにするため、すべてのアイテムと生存者を200ピクセル以上離して設置する。
- **初期位置の距離制限**: ゲーム開始直後、プレイヤーの目の前にクマや脱出口が配置されるのを防ぐため、プレイヤー初期位置からクマおよび脱出口までの距離制限を、最小画面12枚分、最大画面25枚分（画面サイズを基準とした距離換算）とする。

## 2. アイテムシステム
フィールドには以下のアイテムがランダムに配置される。いずれも初期配置数は限定されている。

### 2.1 地図 (出現数: 1個)
- **効果**: 
  - 獲得すると、フィールドのマップ全体を表示できるようになり、山の出口（脱出口）の位置がわかるようになる。
  - 獲得次第、画面上に「地図所持ボタン」が表示される。
  - キーボード of `R` キーを押すことで、地図画面を表示することができる。
- **簡易マップ（全体マップ表示画面）の詳細仕様**:
  - **表示レイアウト**: レトロな緑色の透過型戦術GPSレーダーモニター風のUI。
  - **現在地のプロット**: プレイヤーの現在位置を青色の点滅アイコン「P」でリアルタイムにプロットする。
  - **脱出口のプロット**: 地図上に山の脱出口の位置を緑色の扉アイコン「🚪 (EXIT)」でプロットし、明確に可視化する。
  - **生存者のプロット**: 
    - すでに発見・救出に成功した生存者（GPS同期済）の位置を、対応する名前ラベルと共に明るく明滅するエメラルドグリーンのサークル「🟢」でプロットする。
    - 未発見の遭難者については、通常探索時にはマップにプロットされない（ホラーとしての緊張感を保つ仕様。ただしデバッグ・シミュレータ用の検証機能として、一括表示できる開発トグルを配備する）。
  - **安全ルート表示システム（生存者の助言） [NEW]**:
    - **自動的な案内ライン**: 生存者を救出した際、その場所から「脱出口（EXIT）」までを直線で結ぶ、黄色く点滅する一時ルート（ヒントライン）が簡易マップ上に表示される。
    - **自動開閉連動**: 地図を所持している状態で生存者を救助すると、自動的に簡易マップ（GPS画面）が起動・展開され、救助した遭難者を起点とした脱出ルートが一時描画される。
    - **表示持続時間（自動消滅）**: 表示されてから **10秒間** が経過すると、ラインはフェードアウト（自動的に消滅）する。
    - **手動での再表示（インタラクティブ操作）**: 救出済みの生存者のプロットアイコンをクリック（タップ）することで、いつでもその生存者からの安全脱出ルートを再度10秒間表示させることができる。
- **目的**: ゲームクリアのために必須のアイテム。

### 2.2 鈴 (出現数: 3個)
- **効果**: 
  - 使用すると、10秒間、フィールド上で熊を寄せ付けない（追跡を無効化する）効果を発揮する。
  - 獲得するたびに所持数が 1 ずつ増加する。
  - 簡易マップ（地図）同様に、探索中にキーボードの `Q` キーを押すことで使用可能。
  - **連続消費防止（クールダウン）**: 連続で誤って消費しないよう、効果が持続している10秒間は、再度Qキーを押しても鈴を鳴らせない制限（再使用不可のクールダウン）を設ける。
  - **視覚的エフェクト（ソナー機能）**: 鈴を鳴らすと、効果持続中の10秒間はプレイヤーを中心に円状に広がる「金色の波紋（ソナー風エフェクト）」が繰り返し描画され続け、プレイヤーの頭上に残り持続秒数（例: `🔔 ACTIVE: 9.5s`）がリアルタイム表示されることで、効果中であることを明確に伝える。
- **耐久力（制限・管理仕様）**: 
  - すぐに錆びてしまうため、1つの鈴につき2回使用すると壊れてしまう。
  - **管理仕様（B案）**: 鈴を1個獲得するごとに、全体の「残り使用可能回数」が **+2回** され、共通のカウンターとして加算・管理される。使用するごとにカウンターが 1 減少し、0 になると鈴の効果は使えなくなる。
  - **UI表示**: 探索画面（UI）において、所持している鈴の個数の横に、あと何回使用できるかの「残り使用可能回数（耐久力）」を並べて表示する。

### 2.3 熊撃退スプレー (出現数: 3個)
- **効果**: 
  - クマとの戦闘画面でのみ使用可能な防衛アイテム。
  - 使用すると、クマを一度だけ噴射して遠ざけ、戦闘を離脱して探索画面に戻ることができる。
- **消費**: 
  - 使用するたびに所持数を1減らす。

## 3. キャラクターとNPC

### 3.1 プレイヤー（主人公）
- 山に迷い込み、懐中電灯を頼りに脱出を目指す。

### 3.2 生存者（救出対象NPC）
- **配置数**: フィールド上のランダムな位置に、合計5人配置される。
- **移動**: その場から動かず、直立（固定位置）している。
- **救出システム**:
  - マウス右クリックで話しかけることで会話が発生。
  - 会話終了後、その生存者は「救出完了」状態（無事保護され、同行している扱い）となり、プレイヤーの後方をついてくる、または非表示となり「救出人数カウンター」に加算される。
  - **探索・救出の動機**: 生存者を何人救出して脱出できたかによって、クリア時の評価（ランクやエンディング）が変化する。
- **会話内容とヒント**: プレイヤーの生存率を上げるアドバイスや、山の怪異に関するヒントを話す。
- **必須会話内容（ヒント）**:
  1. 「遠くにいる時は静かにその場を去る方がいい」
  2. 「近くにいる時は熊を見ながらゆっくり後退して！興奮させないのが一番だよ」
  3. 「鈴はクマが寄ってこなくなるがすぐに錆びてしまうため、長くは使えない。」
