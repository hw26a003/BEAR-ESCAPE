# ビジュアル・グラフィック仕様

## 1. アートスタイル

### 1.1 ドット絵（ピクセルアート）
- **スタイル**: レトロで不気味な雰囲気を醸し出すドット絵。
- **フィールドタイル**: 256×256ピクセル、または512×512ピクセルの正方形パーツ（タイル）を組み合わせてフィールドマップを構成する。
  - **仕様特性**: これは1マス単位のマップチップ（床や壁など）ではなく、木々、障害物、道などがまとめられた、複合的な「大オブジェクトブロック（背景パーツ）」として扱う。これらをパズルのように配置・組み合わせることで、効率的に不気味な夜の山道を表現する。
- **キャラクター**: 主人公（SCARED TRAVELER）、クマ（DEMONIC BEAR）、NPCについても、同様の不気味なダークホラー調のドット絵デザインで統一する。

### 1.2 キャラクター・スプライト仕様
キャラクターはすべて、8方向（北、北東、東、南東、南、南西、西、北西）の移動アニメーションおよび特殊なリアクションモーションを持つスプライトアセットで構成される。

#### 1. プレイヤー：怯える旅人（SCARED TRAVELER）
- **ビジュアル特徴**:
  - 緑色のジャケット/コートを着用し、茶色のズボンを履き、茶色いリュックサックを背負った旅人の姿。
  - 乱れた茶髪と、冷や汗をかき恐怖で青ざめ、目を見開いた怯えた表情が特徴。
- **アニメーション仕様**:
  - **移動時 (Run)**: 8方向すべてに対応した、必死に走って逃げる（ランニング）アニメーション。
  - **特殊アクション (SPECIAL - Reaction)**:
    - 恐怖でその場にしゃがみ込み、両手で頭を抱えて激しく怯えるモーション。
    - 地面に四つ這いになって必死に這いつくばる、または尻もちをつく・転倒するモーション。
    - これらは、クマとエンカウントした瞬間やダメージを受けた際、スタミナ切れでへたり込んだ際などの演出・リアクションとして割り当てられる。

#### 2. クマ：実験体ゾンビベア（ZOMBIE BEAR）
- **ビジュアル特徴**:
  - 全身の皮膚や肉が激しく損壊・ゾンビ化した巨大でおぞましい実験体ベアの姿。
  - 背中や足元、側面から赤いのたうつ寄生組織（触手や露出した肉管、筋組織）が何本も這い出ており、リアルタイムに不気味にくねくねと蠢いている。
  - 体毛は泥や血痕で汚れた茶褐色で、皮膚の一部は剥がれ落ちて内側の黒ずんだ壊死組織やグレーの筋組織が露出している。
  - 首には、血と泥に汚れた「スタッズ（金属鋲）付きの不気味な暗青色の実験用首輪」を巻いており、元々実験施設等で拘束されていた痕跡を思わせる。
  - 前足の爪は異常に発達して太く長く、爪先が鮮血で真っ赤に染まっている。
  - 顔面の右側は完全に壊死して眼球が黒い空洞となっており、そこから黒い体液（血涙）が絶え間なく流れ落ちている。一方、左目は不気味に黄色く濁って光る異様な瞳を持つ。
  - 口元は裂け、血まみれの鋭い牙（むき出しの歯列）が不揃いに並び、顎からはしたたる不気味な唾液（ヨダレ）が糸を引いて垂れ下がっている。
- **アニメーション仕様**:
  - **移動時 (Walk)**: 8方向すべてに対応した、重々しくかつ、うにょうにょと動く赤い触手を引きずりながら獲物を執拗に追う歩行アニメーション。
  - **特殊アクション (SPECIAL - All Directions)**:
  - 背中や足元の赤い寄生組織（触手）をより狂気的に激しくのたうち回らせるモーション。
  - 上体を大きくのけぞらせて咆哮し、裂けた口から不気味な紫色の闇の霧、オーラ、またはブレスを吐き出すモーション。
  - これらは、プレイヤーを追跡中（CHASE状態）や戦闘中に強力な攻撃・威嚇行動を行う際のアニメーションとして割り当てられる。

#### 3. 拾得アイテム：魔除けの鈴（BELL）
- **ビジュアル特徴**:
  - アンティーク感のある、重厚な真鍮（ブロンズ・ゴールド）で作られた鐘型の鈴。
  - 上部には古びたロープ（結び紐）が結ばれており、鈴本体と紐を繋ぐ部分に真鍮製の丸いリング状の取っ手がある。
  - 鈴の表面には、波型や植物の葉を模した精密で美しいレリーフ（彫刻文様）が刻まれている。
  - 鈴の開口部（下部）からは、音を鳴らすための丸い「舌（クラッパー・振り子）」が小さく覗いている。

#### 4. 拾得アイテム：撃退スプレー（SPRAY）
- **ビジュアル特徴**:
  - 白と鮮やかな赤のツートンカラーで彩られた本格的な工業用エアゾール缶（スプレー缶）。
  - 円柱状の赤い缶身の中央には、白の力強いフォントで「SPRAY CAN」と印字され、その下には詳細な説明文や警告マーク（可燃性マークなど）が繊細に表現されている。
  - 缶の頂部にはメタリックシルバーの円錐状のショルダードームが配置され、その上に赤色のボタントリガーと、ダークグレーの保護ノズルハウジング、そして噴射ストロー（白色）が取り付けられている。

#### 5. 拾得アイテム：地図（MAP）
- **ビジュアル特徴**:
  - 重厚な革張りの表紙が施された、アンティーク調の豪華な地図帳（フィールドガイド）の意匠。
  - 地図の周囲を囲うフレームには、深緑色と真鍮（ゴールド）をベースに、植物の葉や美しい蔓（ツタ）のレリーフが細部まで刻まれている。
  - 右上には、アンティークな金属製の丸型羅針盤（コンパス）が埋め込まれている。
  - 地図面には「OAKHAVEN TOWN」とその周辺環境が描かれており、森林（グリーン）、川（ブルー）、道路、城のアイコンなどが古風なタッチで表現されている。
  - ガイドの左側には、インク瓶とそれに挿された繊細な金色の羽根ペン（羽ペン）が添えられており、探索と記録への意欲を掻き立てるデザインになっている。

### 1.4 画面演出（ライティング）
- **トーン**: 夜の山道を表現するため、画面全体を暗めに設定する。
- **懐中電灯効果**: プレイヤーキャラクターを中心に、懐中電灯で照らしているような円形の可視領域（光）を表現する。
  - 光の当たっている部分のみグラフィックが鮮明に見え、光の外側は真っ暗（または非常に暗い状態）にする。

## 2. 画面レイアウトと解像度

### 2.1 基本画面サイズ
- **表示解像度**: 960 × 544 ピクセル（アスペクト比 16:9 近似、15:8.5）。
- PC画面での全画面表示、またはアスペクト比固定のウィンドウ表示を想定。

### 2.2 必要な画面遷移
1. **開始前画面（タイトル / プロローグ）**:
   - 物語風のあらすじテキストを表示。
   - 各種アイテム（地図、鈴、熊撃退スプレー）の説明を表示。
   - 「ゲーム開始」ボタンを配置。
2. **フィールド探索画面（メインゲーム）**:
   - 960 × 544ピクセルの探索画面。プレイヤー、NPC、クマ、アイテム、懐中電灯エフェクト。
   - 地図所持時に「地図所持ボタン」を表示。
   - 鈴所持時に「鈴の使用ボタン」と「所持個数」、およびその横に残り使用可能回数（耐久力）を表示。
3. **地図表示画面**:
   - `R` キー入力時にオーバーレイ表示される。
   - フィールド全体の構造と、プレイヤーの現在地、脱出口の位置を示す。
4. **戦闘画面**:
   - クマと接触時に移行する、コマンド選択式の緊迫感ある画面。
   - クマのグラフィック（恐ろしいドット絵）と、3つの選択肢（「走って逃げる」「死んだふりをする」「アイテムを使う」）を表示。
5. **ゲームオーバー画面**:
   - クマに襲われた時の画面。
   - タイトルに戻る、またはチェックポイントからリトライするボタンを配置。
6. **ゲームクリア（脱出成功）画面**:
   - 脱出口に到達した時の画面.
   - 救出した生存者の人数と、それに応じた「評価ランク」および「クリアエンディング（テキスト及び演出）」を表示する。
   - **評価ランクと分岐仕様**:
     - **Sランク (生存者5人全員救出)**: 「奇跡の生還者たち」エンディング。全員で無事に山を降り、最高評価。
     - **Aランク (生存者3〜4人救出)**: 「多くの命を救って」エンディング。多くの犠牲を防ぎ、生存者たちから感謝される。
     - **Bランク (生存者1〜2人救出)**: 「ほろ苦い生還」エンディング。何人かは救えたものの、救えなかった者への悔いが残る。
     - **Cランク (生存者0人救出・自力脱出のみ)**: 「孤独な生還」エンディング。自分だけは辛うじて助かったが、深い喪失感と後悔が残る。
   - タイトルに戻るボタン、スタッフロールの表示。

### 2.3 クマ接近警告メーター（画面右下UI）
フィールド探索画面の右下に、クマとの位置関係と接近度を視覚的に伝える「接近警告メーター（例：鼓動する心臓やレーダーを模したUIアイコン）」を表示する。
- **基本状態（クマが画面外）**:
  - クマがプレイヤーの画面外（一定距離以上離れている状態）にいるときは、メーターは**グレー（無彩色）**で表示され、静止または極めて緩やかなアニメーションを行う。
- **警告状態（クマが画面内・接近時）**:
  - クマがプレイヤーの画面内（またはプレイヤーを感知し追跡を開始した距離）に入ると、メーターが**赤色**に変化し、点滅を開始する。
- **接近度連動アニメーション**:
  - クマとプレイヤーの距離が縮まるほど、メーターの赤色点滅（および鼓動アニメーション）のテンポが動的に加速する。
  - 心音SEのテンポ上昇と同期させることで、視覚と聴覚の双方から「今どれだけ危険か」をプレイヤーに直感的に分からせ、高い緊張感を演出する。

## 3. サウンド・音響演出仕様

不気味で臨場感あふれるホラー体験を提供するため、視覚効果に合わせた音響演出を行う。

### 3.1 距離連動型「心音（鼓動音）」
- **仕様**: プレイヤーとクマの直線距離をリアルタイムに計算し、その接近度（距離）に応じて、バックグラウンドで鳴る「心臓の鼓動音（ドクン、ドクン）」の音量とテンポを動的に変化させる。
- **挙動**:
  - **安全（距離：遠）**: 心音は聞こえない。静まり返った環境音のみ。
  - **警戒（距離：中）**: 低く、遅いテンポで「ドクン……ドクン……」と鳴り始める。
  - **危険（距離：近）**: クマがプレイヤーに近づく（画面内に入るレベル）につれて、音量が大きくなり、テンポ（BPM）が急上昇する（「ドクンドクンドクン！」）。プレイヤーにクマの接近を本能的に察知させ、緊迫感を最大限に高める。

### 3.2 アクション・イベントSE
- **鈴の音**:
  - 探索画面でプレイヤーが鈴を使用した際、静寂を破るような澄んだ「チリン、チリン」という鈴の音が響く。
- **スプレーの噴射音**:
  - 戦闘画面で「熊撃退スプレー」を使用した際、勢いよくガスが噴射される「プシューッ！」という不気味な静寂を破る音が鳴る。
- **懐中電灯のノイズ音**:
  - 懐中電灯のバッテリー残り時間が2分以下になり、光が明滅する演出が開始されると、その点滅に同期して「ジーッ…」「ジジッ…」という不規則な放電・電気ノイズ音が発生する。光源を失う恐怖を聴覚的にも煽る。

### 3.3 環境音（アンビエントサウンド）
- **仕様**: ゲーム開始からクリア・ゲームオーバーになるまでの探索中、バックグラウンドで常に「低くうねる風の音」や「木々を揺らす冷たい吹雪の音」を合成したアンビエント音をループ再生する。
- **演出効果**: 
  - Web Audio API等によるリアルタイム合成ノイズ（ホワイトノイズと周波数を緩やかに変調するバンドパスフィルターの組み合わせ）を用いて、プレイヤーの行動や環境に即した不気味で生々しい風のうねりを表現する。
  - 静寂の中に風鳴りが控えめに鳴り続けることで、極寒の雪山に一人取り残された孤独感と、いつクマが現れるかわからない不穏なホラー空間の没入感を極限まで高める。

## 4. 開発用代替アセット（プロトタイプ）の代用方針

プロトタイプとして早期に動作確認・ゲームバランス調整を行う際、本番用のドット絵や音源アセットがまだ揃っていない場合は、以下の代替方針で実装を進める。

### 4.1 簡易グラフィックによる代替（Canvas描画）
- **床・地形**: グリッドラインや、プログラムによる簡易テクスチャ（緑色の濃淡による山林の表現）。
- **プレイヤー**: 青色の円または簡易的な矩形、向いている方向を示す方向線付きのグラフィック。
- **クマ**: 赤色の大きな円（または矩形）。
- **NPC（生存者）**: 各キャラクター固有のイラスト調ドット絵として精巧に描き分ける。
  - **佐藤 (Sato) [料理人・主婦風]**: ピンクのふんわりとした髪、爽やかな水色のドレスに白いエプロン、木のスプーンと銅の片手鍋を持った温厚な女性。
  - **鈴木 (Suzuki) [修道士風]**: 禿げ上がった頭（トンスラ）に白ヒゲ、茶色の修道ローブをまとい、分厚い革装の本を抱え、長い木製の杖を持った老人。
  - **高橋 (Takahashi) [弓使い風]**: 赤褐色の髪を一本の三つ編みに結い、茶色のレザー製アーマー（狩人服）を着用、背中には矢筒を背負い、木製のロングボウ（弓）を左手に携えた女性。
  - **田中 (Tanaka) [鍛冶屋風]**: 茶色の豊かな顎ひげ、ブラウンの頑丈なレザーエプロンを身につけ、巨大な鍛冶屋ハンマーと木製の丸盾を持った屈強な男性。
  - **伊藤 (Ito) [狩人・盗賊風]**: 深く被った濃紺のフード付きケープ（マント）、ダークグレーの戦闘服、手には手斧とナイフを持ったミステリアスな風貌の人物。
- **アイテム・障害物**: 
  - 木・岩：茶色やグレーの円・多角形（衝突判定範囲を可視化する円付き）。
  - アイテム：黄色や白の小さな四角形、または各アイテムのイニシャル文字（「D」（地図）、「B」（鈴）、「S」（スプレー）など）。
  - 脱出口：緑色のドア型の四角形。

### 4.2 Web Audio APIによる合成音による代替（SE・BGM）
- **アクションSE（鈴・スプレーなど）**: Web Audio APIのオシレーター（発振器）から生成する、矩形波やサイン波を組み合わせた簡易的なピコピコ音（ビープ音）。
  - 例：鈴（チリン）＝高音のサイン波の短いパルス。
  - 例：スプレー（プシュー）＝ホワイトノイズにローパスフィルターをかけた減衰音。
- **心音SE**: 超低音（50Hz〜80Hz）のサイン波を、プレイヤーとクマの距離に連動した間隔（テンポ）で「ドクン、ドクン」と短いエンベロープで発音させる。
- **環境音（アンビエント）**: ホワイトノイズに、LFO（低周波発振器）で周波数を揺らしたバンドパスフィルターを適用し、不気味にうねる風の音を動的にリアルタイム合成する。
